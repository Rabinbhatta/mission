Edited image

Here are the steps how I edited the image:

  1. First, click on the File and new and create a canvas of size 600*400px
  2. Now again click on the file and open option to select the image.Repeat the same step to open other image.
  3. Now copy the backgroup image in the canvas you created.
  4. To mirror the image click on layer and then on tranform and then flip horizontal.Do this for both image.
  5. Now select elicipse selection and crop the image and then click select feather then giver radius of the feathure.
  6. Then copy the image and paste it in the background image and move it in the right position.
  7. Now create a new canvas and use Rectangle select tool and in the left You can also see rounded corners option to select rounder rectangle corner.
  8. Crop the canvas and paste it in the background image
  9. Now in new canvas using text tool write your student id no and coppy and paste it in the rectangular box in background image.
